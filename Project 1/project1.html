Satellite Tracking and Trajectory Prediction Application

This project aims to enhance Space Situational Awareness (SSA) by providing real-time satellite tracking and trajectory prediction.

The application leverages open-source Two-Line Element (TLE) data and integrates traditional mathematical models along with an experimental Long Short-Term Memory (LSTM) machine learning model for precise satellite trajectory prediction.

Features:

  • Real-time satellite tracking.
  • Prediction of satellite trajectory using machine learning.
  • Utilization of TLE data for calculations.

Technologies Used:

  • Python
  • Machine Learning (LSTM Model)
  • JavaScript for visualization
  • Web frameworks for front-end
